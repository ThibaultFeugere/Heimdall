# Heimdall

## Membres du projet

- FEUGERE Thibault
- MOURET Vivien
- SIX Hugo

## Description du projet

Heimdall est un jeu de combat tour par tour accompagné d'une musique scandinave.

### Utilisateur

- Se connecter
- S'enregistrer
- Combattre
- Statistiques
- Boutique

### Combat

- Attaquer
- Soigner
- Abandonner

### Dieu

- Points de vie
- Expérience

### Boutique

- Potion de vie
- Armes
